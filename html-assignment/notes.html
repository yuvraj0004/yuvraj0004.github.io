HTML Tags with Description

Ordered List tag(ol)
This tag is basically used to list our content in a particular order.
Unordered List Tag(ul)
This tag is basically used to list our content in no particular order that is by showing bullet points right before our content.
The Paragraph element(p)
This tag is used to define a paragraph in website.
The Hyperlink Tag
This tag is used to link from one page to another.
The Content Span element
This tag is used to mark up a part of a text, or a document.
